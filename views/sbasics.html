SEO Basics

The difference between SEO and SEM is simply that Search Engine Optimization is part of Search engine marketing or Search marketing as it is widely known. Both processes aim in increasing visibility in search engines.


SEO Strategies

1. Domain Name Strategy: Using domain name services sets your business up for success where it all begins, your websites name. -A domain name strategy can add incredible value to the appearance of your website by taking advantage of the most visible component, the address. -You also need to establish and safeguard your domains so you have a secure presence online.
2. Linking Strategy: Links are an incredibly important factor to improve search performance, and they seem to be making a resurgence this year. If you have never executed a link building strategy its important to know all the facts, because link building is inherently risky, and only the highest quality of tactics are recommended. These resources will help beginners understand a bit more; it pays to read them before executing any campaign.
3. Title Tag Strategy: Title tags are pretty much exactly what they sound like the title of the page. As a title, this tag should be used to describe what a user can expect to find on a particular page.When writing title tags with SEO in mind, your first goal should be to accurately describe the content of the page in question whether its a product, collection or informational page. Its also vital to find ways to naturally weave in relevant keywords into the tag whenever possible.
4. Meta Description Tags Strategy: A meta description is a type of meta tag used in HTML to describe what the web page is about. It appears as a quick snippet on the SERPs right beneath the link. Meta descriptions are not a ranking factor per se, as they have a very minimal impact on rankings, though they may have some impacts with personalized search results. Their main function is telling searchers what your web page is about. Having compelling meta descriptions aids in the click-through rate to your site. Think of your meta descriptions as a sales pitch for searchers, not as ranking factors for crawlers.
5. Alt Tags Strategy: The Alt tag is an older HTML coding strategy originally envisioned to compensate for users and browsers not equipped to handle images. And while all of todays web browsing includes many types of graphics, the alt tag remains an important part of making a site accessible to all users. The alt tag is a descriptive line of HTML code that describes an image on a web page so that the user knows what the image represents if the picture doesnt display correctly.
6. Search Engine Indexing Strategy: The indexing mechanism is latent semantic indexing. In latent semantic indexing search engines, determine the relationship between different entities or part of a blog content. By understanding this relationship between different entities, search engines understand the context meaning of the blog content easily. Once the content is understood properly by search engines, it is easy for them to return accurate results for long tail keywords.
